# Escaping-The-Fire-3D-Simulation
A 3D fire safety simulation game developed using Unity, Blender, and C#. Players navigate hazardous environments, avoid fire and smoke, and reach safety before health is depleted.

## Overview

Escaping The Fire is a 3D fire safety simulation game developed for educational purposes. Players must navigate a burning environment, avoid fire and smoke hazards, manage their health, and find the exit before it is too late.

## Features

- 3D environment exploration
- Fire and smoke hazards
- Health system
- Flashlight mechanics
- Dynamic collision detection
- Win and lose conditions
- Emergency evacuation simulation

## Gameplay

1. Start the game
2. Navigate the environment
3. Avoid fire hazards
4. Use the flashlight
5. Monitor health status
6. Reach the exit safely

## Technologies Used

- Unity
- Blender
- C#
